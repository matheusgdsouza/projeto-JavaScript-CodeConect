# CodeConnect 🚀

Projeto desenvolvido com **HTML, CSS e JavaScript**, com foco na prática de **assincronismo em JavaScript**, utilizando **Promises**, **async/await** e manipulação dinâmica do **DOM**.

A aplicação simula o carregamento e exibição de dados de forma assíncrona, explorando conceitos essenciais do desenvolvimento front-end moderno.
